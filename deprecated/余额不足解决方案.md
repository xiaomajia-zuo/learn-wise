# 账户余额不足解决方案

当遇到 "402 Insufficient Balance" 错误时，说明您的AI服务账户余额不足。

## 解决方案

### 方案1：充值DeepSeek账户（推荐）

1. 访问 [DeepSeek平台](https://platform.deepseek.com)
2. 登录您的账户
3. 进入充值页面
4. 选择合适的充值金额
5. 完成支付后即可继续使用

### 方案2：切换到其他AI服务

如果您不想充值，可以切换到其他AI服务提供商：

#### 使用通义千问（阿里云）

1. 访问 [通义千问控制台](https://dashscope.aliyun.com/)
2. 注册/登录阿里云账号
3. 开通通义千问服务并获取API Key
4. 修改 `.env` 文件：
   ```env
   AI_PROVIDER=qwen
   QWEN_API_KEY=你的通义千问API密钥
   ```
5. 重启服务器

#### 使用智谱GLM

1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 注册账号并获取API Key
3. 修改 `.env` 文件：
   ```env
   AI_PROVIDER=zhipu
   ZHIPU_API_KEY=你的智谱API密钥
   ```
4. 重启服务器

#### 使用Moonshot（月之暗面）

1. 访问 [Moonshot平台](https://platform.moonshot.cn/)
2. 注册账号并获取API Key
3. 修改 `.env` 文件：
   ```env
   AI_PROVIDER=moonshot
   MOONSHOT_API_KEY=你的Moonshot_API密钥
   ```
4. 重启服务器

2. 建议设置使用限额，避免意外产生高额费用
3. 定期检查账户余额，及时充值
