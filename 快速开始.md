# 快速开始指南

## 第一步：设置后端

本项目的后端实现：使用 Python + FastAPI（位于 `backend/`）
启动说明：


1. 进入项目根并创建虚拟环境（可选但推荐）：
```bash
cd xxx/learnwise-backend
python3 -m venv .venv
source .venv/bin/activate
```

2. 安装依赖：
```bash
pip install -r backend/requirements.txt
```

3. 在 `backend/.env` 中配置必要环境变量（示例）：
```env
AI_PROVIDER=openai
OPENAI_API_KEY=sk-...
OPENAI_MODEL=gpt-4o-mini
UPLOAD_DIR=backend/uploads
MAX_FILE_SIZE=209715200
```

4. 启动服务（开发模式，自动热重载）：
```bash
uvicorn backend.main:app --reload --host 0.0.0.0 --port 3001
```

5. 验证：
```bash
curl http://127.0.0.1:3001/health
curl http://127.0.0.1:3001/api/ai/config
```

后端将在 http://localhost:3001 运行

## 第二步：设置前端

1. 进入前端目录：
```bash
cd frontend
```

2. 安装前端依赖：
```bash
npm install
```

3. （可选）创建 `.env` 文件配置API地址：
```
REACT_APP_API_URL=http://localhost:3001/api
```

4. 启动前端应用：
```bash
npm start
```

前端将在 http://localhost:3000 运行

## 使用说明

1. 打开浏览器访问 http://localhost:3000
2. 点击"上传学习资料"按钮，选择要学习的文件
3. 上传后会自动生成摘要（仅文本和代码文件）
4. 在右侧AI对话框中可以提问关于学习资料的问题

## 支持的文件格式

- **视频**：.mp4, .webm, .ogg, .mov
- **文本**：.txt, .md, .pdf, .epub
- **代码**：.js, .py, .java, .cpp, .c, .html, .css, .json, .xml

## 常见问题

### AI API Key 获取（国内用户推荐）
- **DeepSeek**（推荐）：访问 https://platform.deepseek.com/ 注册并获取API密钥
- **通义千问**：访问 https://dashscope.aliyun.com/
- **智谱GLM**：访问 https://open.bigmodel.cn/
- 详细配置说明请查看 `国内AI服务配置指南.md`

### OpenAI API Key（需代理）
如果需要使用OpenAI，访问 https://platform.openai.com/api-keys 创建API密钥

### 端口冲突
如果3001或3000端口被占用，可以修改：
- 后端：修改 `.env` 中的 `PORT`
- 前端：修改 `package.json` 中的启动脚本添加 `PORT=3002`

### 视频无法播放
确保视频文件格式为支持的格式（mp4推荐），且文件大小在限制范围内
