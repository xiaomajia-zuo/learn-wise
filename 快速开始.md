# 快速开始指南

## 第一步：设置后端

1. 安装后端依赖：
```bash
npm install
```

2. 创建 `.env` 文件（复制以下内容）：
```
PORT=3001
AI_PROVIDER=deepseek
DEEPSEEK_API_KEY=sk-48b0ff953d354658a94c9cce8007f615
UPLOAD_DIR=uploads
MAX_FILE_SIZE=104857600
NODE_ENV=development
```

**重要**：国内用户推荐使用DeepSeek等国内AI服务，详见 `国内AI服务配置指南.md`

3. 启动后端服务器：
```bash
npm start
# 或开发模式（自动重启）
npm run dev
```

后端将在 http://localhost:3001 运行

## 第二步：设置前端

1. 进入前端目录：
```bash
cd frontend
```

2. 安装前端依赖：
```bash
npm install
```

3. （可选）创建 `.env` 文件配置API地址：
```
REACT_APP_API_URL=http://localhost:3001/api
```

4. 启动前端应用：
```bash
npm start
```

前端将在 http://localhost:3000 运行

## 使用说明

1. 打开浏览器访问 http://localhost:3000
2. 点击"上传学习资料"按钮，选择要学习的文件
3. 上传后会自动生成摘要（仅文本和代码文件）
4. 在右侧AI对话框中可以提问关于学习资料的问题

## 支持的文件格式

- **视频**：.mp4, .webm, .ogg
- **文本**：.txt, .md
- **代码**：.js, .py, .java, .cpp, .c, .html, .css, .json, .xml

## 常见问题

### AI API Key 获取（国内用户推荐）
- **DeepSeek**（推荐）：访问 https://platform.deepseek.com/ 注册并获取API密钥
- **通义千问**：访问 https://dashscope.aliyun.com/
- **智谱GLM**：访问 https://open.bigmodel.cn/
- 详细配置说明请查看 `国内AI服务配置指南.md`

### OpenAI API Key（需代理）
如果需要使用OpenAI，访问 https://platform.openai.com/api-keys 创建API密钥

### 端口冲突
如果3001或3000端口被占用，可以修改：
- 后端：修改 `.env` 中的 `PORT`
- 前端：修改 `package.json` 中的启动脚本添加 `PORT=3002`

### 视频无法播放
确保视频文件格式为支持的格式（mp4推荐），且文件大小在限制范围内
